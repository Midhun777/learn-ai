An Intelligent Personalized Learning Roadmap Ecosystem

Section 1

Project Abstract

EduPath AI is an innovative digital ecosystem designed to bridge the gap between aspirational learning goals and structured educational pathways. In an era of informational overflow, learners often struggle to identify the most efficient sequence of study for complex skills. This project presents a full-stack solution that leverages Generative Artificial Intelligence (Gemini API) to curate personalized, high-quality learning roadmaps.

The system incorporates three distinct user tiers: Administrators, who oversee platform health and resource validation; Instructors, who contribute specialized content and monitor roadmap effectiveness; and Students, who engage in personalized learning journeys.

Key features include dynamic roadmap generation, resource curation with estimated timelines, progress tracking, and automated certification upon milestone completion. Built on a modern MERN (MongoDB, Express, React, Node.js) architecture, EduPath AI ensures scalability, responsiveness, and a premium user experience tailored for the modern academic environment.

Database Structure

TABLE: tbl_users

Stores primary authentication and profile information for all system users.

Sl no Field Type Description
1user_idINTUnique identifier for the user (Primary Key).
2full_nameVARCHAR(100)Complete name of the user.
3email_addrVARCHAR(150)Official email address (Unique).
4pwd_hashVARCHAR(255)Encrypted password string.
5user_roleVARCHAR(20)'Admin', 'Instructor', or 'Student'.

TABLE: tbl_roadmaps

Contains details of the AI-generated or instructor-curated learning paths.

Sl no Field Type Description
1roadmap_idINTUnique roadmap identifier (Primary Key).
2skill_nameVARCHAR(100)Name of the skill or subject.
3difficulty_lvlVARCHAR(50)Beginner, Intermediate, Advanced.
4created_byINTCreator's user ID (Foreign Key).
5category_idINTCategorical grouping ID (Foreign Key).

TABLE: tbl_resources

Stores individual learning assets associated with specific roadmaps.

Sl no Field Type Description
1res_idINTUnique resource identifier (Primary Key).
2res_titleVARCHAR(200)Title of the learning material.
3res_urlTEXTDirect link to the resource.
4roadmap_idINTRoadmap ID (Foreign Key).
